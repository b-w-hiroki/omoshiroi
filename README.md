# ゲームを題材に「面白い」とは何かを考える
**感情の運動としてのゲームデザイン**

---

## 起動方法

ダブルクリックまたはブラウザで `index.html` を開くだけで動作します。
ローカルサーバーは不要です。

```bash
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

---

## 操作方法

| 操作 | 動作 |
|------|------|
| `→` / `↓` / `Space` | 次のスライドへ |
| `←` / `↑` | 前のスライドへ |
| `Home` | 最初のスライド |
| `End` | 最後のスライド |
| スワイプ（タッチ対応） | 左スワイプ→次、右スワイプ→前 |

---

## PDF出力方法

1. Chrome または Edge でブラウザを開く
2. `Cmd + P`（macOS）/ `Ctrl + P`（Windows）で印刷ダイアログを開く
3. 以下の設定にする
   - **送先**：PDF に保存
   - **用紙サイズ**：A4
   - **向き**：横
   - **余白**：なし
   - **背景のグラフィック**：✅ オンにする（重要）
4. 「保存」をクリック

> Chrome 推奨。Safari では背景色が正しく出力されない場合があります。

## 編集時の注意点

- **スライドの追加・削除**：`index.html` 内の `<section class="slide">` を追加/削除するだけで自動的に枚数が更新されます
- **色の変更**：`styles.css` 冒頭の `:root {}` ブロック内の CSS 変数を変更してください
- **フォント変更**：CSS 変数 `--font` を書き換えてください
- **スピーカーノート**：各スライド内の `<!-- Speaker note: ... -->` コメントに記載しています
- **外部依存なし**：CDN・外部画像・外部フォントを一切使用していません。オフライン環境でも動作します
